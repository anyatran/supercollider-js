# Supercollider+Javascript

## Run Threejs
``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```

## Run SC UDP server
``` bash
# Install dependencies in web as well
cd ./web && npm install

# Run the local server at localhost:8080
node .
```
